# email_playground_tools
Some code samples to test the possibilities of automation in Python.
Algumas amostras de código para testar as possibilidades de automação em Python.


Shooter: É o código responsável por criar o e-mail, consumindo uma API que gera dados falsos de CPF e Nome Completo.
Sender: É o código responsável por realizar a conexão SMTP e enviar o e-mail a um destinatário fixo e suas cópias.
